**Customer Churn — Exploratory Data Analysis**

This project performs an end-to-end exploratory data analysis (EDA) on a telecom customer churn dataset to uncover the key factors driving customer attrition. The dataset includes customer demographics, account information (tenure, contract type, payment method), and subscribed services (phone, internet, streaming, security add-ons), along with a binary churn label.

The analysis covers data cleaning (handling missing/blank values in `TotalCharges`, encoding categorical fields), data quality checks (duplicates, nulls), and visual exploration using Matplotlib and Seaborn — including count plots, histograms, and pie charts — to examine how churn varies across gender, senior citizen status, tenure, contract type, internet/add-on services, and payment method.

Key findings show that customers on month-to-month contracts, with fiber optic internet, low tenure, and electronic check payments are significantly more likely to churn, while customers with longer contracts and add-on services like online security and tech support tend to stay longer. These insights lay the groundwork for building a predictive churn model to help the business proactively target at-risk customers with retention strategies.

**Tech stack:** Python, Pandas, NumPy, Matplotlib, Seaborn
